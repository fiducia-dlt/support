User Accounts

The following section describes how you can get user accounts in TAG TrustNet.

New Node

When you first sign up to be a TAG TrustNet member, all people who have requested an account will be sent an email from TAG TrustNet with the user name, the URL to login, the node name, and a way to set their password.

Existing Node

If you have an existing node on TAG TrustNet and want to request an account for another person, please have the person who is the primary contact with TAG TrustNet send an email with this request to support@tagtrust.net. This email needs to include the following information for the person needing the account:

  • first and last name
  • email address.

Reset your password

If you have a TAG TrustNet account and forgot your password, please contract support@tagtrust.net.
